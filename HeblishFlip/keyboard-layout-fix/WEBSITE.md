# Website URL for Buy Me a Coffee & stores

The extension **does not have a live URL until you publish it**. For Buy Me a Coffee you need **any** public page about the project.

## Easiest options (pick one)

### Option A — GitHub Pages (recommended, free)

1. Create a GitHub repo (e.g. `heblish`)
2. Upload the `website/` folder from this project
3. Settings → Pages → deploy from `main` / folder `website`
4. Your URL: `https://YOUR_GITHUB_USERNAME.github.io/heblish/`
5. Use that URL in Buy Me a Coffee **and** in store listings

### Option B — Store listing (after publish)

- Edge: `https://microsoftedge.microsoft.com/addons/detail/...`
- Chrome: `https://chromewebstore.google.com/detail/...`

Use the store page as your “website” once live.

### Option C — Buy Me a Coffee only

Some creators use their BMC page as the linked site temporarily:
`https://buymeacoffee.com/YOUR_USERNAME`

---

Replace `YOUR_USERNAME` in `website/index.html`, `popup.js`, and `STORE_LISTING.md`.
